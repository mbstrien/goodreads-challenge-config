# goodreads-challenge-config

Remote config and live challenge data for the Goodreads Challenge Recommender Chrome extension.

## Setup

1. **Enable GitHub Pages**: Settings → Pages → main / root
2. **Add Apify secret**: Settings → Secrets → `APIFY_TOKEN`
3. **Manual refresh**: Actions → Refresh challenge data → Run workflow

The live URL will be:
`https://mbstrien.github.io/goodreads-challenge-config/challenges.json`
